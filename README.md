# Server


## How to start


1. Install the dependencies

```bash
npm install
```

2.Install migration

```bash
npx prisma migrate dev
```

3. Start the development server

```bash
npm run start:dev
```

4. Run unit-test

```bash
npm run test
```

# Client


## How to start


1. Install the dependencies

```bash
npm install
```

2. Start the development server

```bash
npm start
```


### Quick start
```bash
./start-server.sh
```
